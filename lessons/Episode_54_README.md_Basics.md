# Episode 54: README.md Basics

- **README.md** stands for "Read Me" and is written in Markdown.
- Markdown syntax:
  - Headings: `#` for h1, `##` for h2, up to `######` for h6.
  - Horizontal rule: `---` or `___`.
  - Paragraphs: Plain text or `<p>text</p>`.
  - Inline code: `` `code` ``.
  - Code block: 
    ```markdown
    ```language
    code
    ```
    ```
  - Ordered list:
    ```markdown
    1. Item
    2. Item
       1. Sub-item
    ```
  - Unordered list:
    ```markdown
    - Item
    - Item
      - Sub-item
    ```
